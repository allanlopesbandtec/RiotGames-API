Bean:

é uma classe cujo a instancia é gerenciada pelo SpringBoot!


@Component (classe) --> Vai tornar a classe um Bean
--> Mais genérico, qualquer tipo de classe


@Service
--> Classes de regra de negócio

@Repository
--> Transações com o banco de dados

@Controller
--> Recebe as requisições 


--> Quando a gente fala que a classe é um bean, deixamos a classe "injetavel"

--> para injetar usamos o @Autowired, o Spring cria e destrói a instância da classe
